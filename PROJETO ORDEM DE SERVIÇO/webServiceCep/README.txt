criei um classe que é uma ferramenta para usar o site de vcs

a) Pasta doc:	contém toda a documentação da classe.
b) Pasta lib:	contém a biblioteca dom4j-1.6.1.jar necessaria para ler os XML.
c) pasta src:	contendo o arquivo fonte WebServiceCep.java dentro da pasta java
		e o arquivo Teste.java dentro da pasta sample, apenas um exemplo de uso.

d) pacote webServiceCep.jar:	Contem o arquivo de consulta de CEP, aconselhavel para 
				quem já tem a lib dom4j-1.6.1.jar

e) pasta bin:	contém o pacote dom4j-1.6.1.jar + a classe WebServiceCep.java em um único
		pacote dom4j-plus-webServiceCep.jar, aconselhavel para quem não tem o dom4j
